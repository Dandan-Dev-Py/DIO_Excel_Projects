# DIO_Excel_Projects
Aqui neste repositorio estarão os projetos que desenvolvi para incrementar e formar meu portfolio de ferramentas de planilha por conta da minha aventura no curso de Excel com inteligencia artificial da DIO.

## 1 - Ferramenta de simulação de investimentos no Excel (DanDan Banks)
este primeiro projeto foi feito usando apenas excel e eu fiz uma logo utilizando canva. O projeto consiste em uma Planilha com o objetivo de auxiliar as pessoas a projetar seus investimentos, com algumas simulações e sugestões de investimento, foram utilizadas imagens, tabelas, formulas (principalmente SOMA e PROCV) e graficos nesse projeto. 
#### * O projeto te permite adicionar seus ganhos mensais e:
- Dá sugestões de valor e investimentos com base no seu perfil de investidor
- Sugere valor seguro a se investir com base em seu salario informado
- E permite ver previsões de ganhos futuros baseados em estimativas pré-determinadas (planilha feita com base no ano de 2026, correções monetarias futuras não foram consideradas)

É uma otima plataforma para quem esta iniciando nos investimentos e não sabe tão bem por onde começar.

## 2 - Ferramenta de organização para declaração de imposto de renda (Tiger Tax)
Este é o segundo projeto desenvolvido por mim, se chama Tiger Tax (Imposto do Tigre) e ela se trata de uma ferramenta composta para te auxiliar a fazer sua declaração de imposto de renda, junto a seu contador ou por conta propria. Esse projeto foca na facilidade e acessibilidade do usuario, sendo fluido, intuitivo e simples, tentando imitar o layout de uma pagina de site em uma planilha excel. para este projeto foi usado unicamente excel, sinto que estou dominando cada vez mais a ferramenta.
(LEMBRE-SE DE CRIAR UMA COPIA DA PLANILHA ANTES DE PREENCHER QUALQUER COISA PARA QUE NINGUEM POSSA ACESSAR SUAS INFORMAÇÕES)

#### * O projeto parece um formulario onde você deve preencher:
- As informações pessoais do titular
- As informações bancarias do titular
- Os recebimentos em conta do titular

Ao final, você terá um formulario organizado e pronto para ser analisado.

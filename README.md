# DIO_Excel_Projects
Aqui neste repositorio estarão os projetos que desenvolvi por conta da minha aventura no curso de Excel com inteligencia artificial da DIO

## 1 - Ferramenta de simulação de investimentos no Excel
este primeiro projeto foi feito usando apenas excel e eu fiz uma logo utilizando canva. O projeto consiste em uma Planilha com o objetivo de auxiliar as pessoas a projetar seus investimentos, com algumas simulações e sugestões de investimento, foram utilizadas imagens, tabelas, formulas e graficos nesse projeto.

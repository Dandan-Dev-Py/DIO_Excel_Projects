## 1 - Ferramenta de simulação de investimentos no Excel (DanDan Banks)
este primeiro projeto foi feito usando apenas excel e eu fiz uma logo utilizando canva. O projeto consiste em uma Planilha com o objetivo de auxiliar as pessoas a projetar seus investimentos, com algumas simulações e sugestões de investimento, foram utilizadas imagens, tabelas, formulas (principalmente SOMA e PROCV) e graficos nesse projeto. 
#### * O projeto te permite adicionar seus ganhos mensais e:
- Dá sugestões de valor e investimentos com base no seu perfil de investidor
- Sugere valor seguro a se investir com base em seu salario informado
- E permite ver previsões de ganhos futuros baseados em estimativas pré-determinadas (planilha feita com base no ano de 2026, correções monetarias futuras não foram consideradas)

É uma otima plataforma para quem esta iniciando nos investimentos e não sabe tão bem por onde começar.

Ao final, você terá um formulario organizado e pronto para ser analisado.

## 3 - Dashboard de vendas do Xbox 
Este é um dashboard de vendas feito para ver o desempenho de vendas da Xbox, ele foi idealizado usando perguntas de negocio para medir diretamente o que era util para a tomada de decisão, usando as habilidades cultivadas no curso de excel. Durante o processo para chegar a criação do meu primeiro dashboard com excel aprendi diversos processos de analise de dados como :

- Storytelling (usado para melhorar a apresentação e interpretação das informações para aqueles que a utilizarão)
- ETL com Power Query (Extract, Transform, Load | Processo de limpeza de dados para uma melhor analise)
- KPIs (Key Perform Indicators) e OKRs (Objectives and Key Results)

E nesse projeto especificamente foram utilizadas as informações extraidas para responder as perguntas:

- Qual o faturamento total de vendas de planos anuais (contendo todas as assinaturas agregadas)
- Qual faturamento total de vendas de planos anuais, separado por autorenovação não tem autorenovação
- Total de vendas de assinaturas do EA play
- Total de vendas de assinaturas com minecraft season pass
- (Extra) - QUAL O PLANO MAIS VENDIDO

Esse projeto me ensinou muito sobre como usar dados para auxiliar na tomada de decisão, no que se é esperado de uma dashboard, quais os pontos importantes a serem analisados, como utilizar as perguntas de negocio e sobre bons habitos com uma dashboard.
